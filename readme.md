## 🔥 Instruções de uso

1. Faça um clone desse repositório;

2. Entre na pasta `web`;

3. Rode o comando `yarn install` para instalar as dependências;

4. Rode o comando `yarn start` para inicializar o servidor em desenvolvimento;
